# Security Policy

BookTrans Desk is a local-first alpha. Do not include API keys, Authorization headers, proprietary EPUB content, or private provider logs in public reports.

See [docs/SECURITY.md](docs/SECURITY.md) for the full project security model.

For `v0.2.4-alpha-stabilization`, diagnostic bundles include a safety summary and remain designed to exclude original EPUB files, exported EPUB files, API keys, Authorization headers, and full book text.

For now, report suspected vulnerabilities through a private maintainer channel if available, or open a minimal public issue without sensitive details.
