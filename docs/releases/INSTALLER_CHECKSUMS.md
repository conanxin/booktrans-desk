# Installer Checksums

Checksums help users confirm that a downloaded installer matches the artifact produced by the release publisher.

## Windows PowerShell

```powershell
Get-FileHash .\BookTrans-Desk-Setup.exe -Algorithm SHA256
```

## Linux/macOS

```bash
shasum -a 256 ./BookTrans-Desk.AppImage
```

## Release Page Instructions

1. Build the package locally.
2. Compute SHA256 checksums for each uploaded artifact.
3. Paste the checksum values into the GitHub Release body.
4. Keep checksum text next to the matching filename.

The repository does not commit installer packages, unpacked release directories, generated EPUB files, or checksum files produced for one local build. Checksums should be generated by the publisher after packaging and pasted into the GitHub Release.
