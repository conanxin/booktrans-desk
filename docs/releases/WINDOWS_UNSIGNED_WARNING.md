# Windows Unsigned Warning

Current alpha Windows builds are unsigned.

Windows SmartScreen may warn that the publisher is unknown. This warning is expected for the current alpha packaging status.

Users should:

- Download packages only from the project GitHub Release page.
- Verify checksums when release checksums are provided.
- Build from source if they prefer not to run a prebuilt alpha package.
- Keep Windows security features enabled.

Users should not disable system security features to run unknown files from unknown sources.

Future release planning may include code signing once the project is ready for broader distribution.
