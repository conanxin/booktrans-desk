# v0.2.5-alpha-rc

## Summary

`v0.2.5-alpha-rc` is a targeted EPUB compatibility release candidate for BookTrans Desk alpha.

## What Changed Since v0.2.4

- Added six synthetic EPUB regression fixtures for real-world structure boundaries.
- Improved chapter import boundaries for nav files, non-linear spine items, URL-encoded manifest hrefs, and EPUB-internal path separators.
- Added reader compatibility notes for manual release candidate checks.
- Strengthened release checks for RC docs and compatibility matrix coverage.

## New Synthetic Fixtures

- `nested-sections.epub`
- `split-text-inline.epub`
- `entities-special-chars.epub`
- `nav-landmarks.epub`
- `duplicate-hrefs.epub`
- `large-chapter-chunking.epub`

## Compatibility Improvements

- EPUB 3 nav files marked with `properties="nav"` are not imported as translation chapters.
- Spine entries with `linear="no"` are skipped during chapter import.
- Manifest hrefs are URL-decoded for zip entry lookup.
- Duplicate filenames in different resource directories are preserved through export.

## Security/Privacy Status

- No commercial EPUB fixtures are committed.
- No generated EPUB outputs or release artifacts are tracked.
- Diagnostic bundles remain redacted and exclude original EPUB files, exported EPUB files, API keys, Authorization headers, and full book text.
- `npm run labels:print` remains print-only.

## Known Limitations

- Windows alpha builds are unsigned.
- macOS packaging is not configured.
- Fixed-layout EPUB, DRM EPUB, PDF, MOBI, and AZW3 remain unsupported.
- Reader compatibility checks are manual.

## Recommended Manual Tests Before GitHub Release

- Run `npm run build`, `npm test`, `npm audit`, and `npm run release:check`.
- Run `npm run pack` and launch the unpacked app if the platform allows it.
- Export synthetic fixture EPUBs and open at least one in Calibre or Thorium Reader.
- Confirm diagnostic bundle copy and release notes are accurate.
