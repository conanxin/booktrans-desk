# v0.2.4-alpha-stabilization

BookTrans Desk `v0.2.4-alpha-stabilization` is a release readiness pass for the open source alpha.

## Highlights

- Adds an alpha release checklist for repeatable pre-release verification.
- Adds diagnostic bundle safety summary text in both the app UI and exported diagnostic bundle.
- Adds optional GitHub label sync instructions without requiring GitHub API access.
- Strengthens release checks for version consistency, release notes, alpha warning text, required docs, and label JSON validity.

## Verification

Before tagging this release, run:

```bash
npm run build
npm test
npm audit
npm run release:check
npm run labels:print
npm run pack
git diff --check
```

## Alpha Warning

This remains an alpha for technical testers. Keep backups of original EPUB files, avoid public uploads of copyrighted content, and review provider privacy terms before translating private books.

## Known Limits

- Windows packages are unsigned.
- macOS packaging is not configured.
- Reflowable EPUB is the supported target; fixed-layout EPUB, DRM EPUB, PDF, MOBI, and AZW3 remain out of scope.
