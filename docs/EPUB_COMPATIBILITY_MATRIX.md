# EPUB Compatibility Matrix

All fixtures listed here are generated by project code in `tests/helpers/createEpubFixtures.ts`. They contain only minimal synthetic text and resources created for this project. No commercial EPUB files are stored in the repository.

| Category | Fixture | Validator | Text-node translation | Export roundtrip | External EPUBCheck | Notes |
| --- | --- | --- | --- | --- | --- | --- |
| Minimal EPUB 3 | `minimal-epub3.epub` | PASS | PASS | Not targeted | Optional | Minimum legal baseline with nav |
| EPUB 2 NCX | `epub2-ncx.epub` | PASS/WARNING accepted | PASS | Not targeted | Optional | EPUB 2 style package with `toc.ncx` |
| Images and CSS | `images-and-css.epub` | PASS | PASS | PASS | Optional | Confirms non-text resources survive export |

| Fixed-layout EPUB | None | Not supported | Not supported | Not supported | User testing only | Out of current scope |
| DRM EPUB | None | Not supported | Not supported | Not supported | User testing only | Encrypted content cannot be processed |
| PDF/MOBI/AZW3 | None | Not supported | Not supported | Not supported | Not applicable | Out of current scope |

Current engineering target is reflowable EPUB. Real commercial EPUB compatibility must be tested by users with their own legally obtained files and should not be committed to this repository.

When reporting compatibility issues for `v0.2.4-alpha-stabilization`, provide validation reports, external EPUBCheck output, diagnostic bundles, and the smallest copyright-safe fixture possible. Do not upload commercial EPUB files or exported translated EPUB files.
