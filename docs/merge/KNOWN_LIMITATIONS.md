# Known Limitations

- PDF support remains HOLD for public release.
- No OCR.
- No DRM handling or DRM bypass.
- No exact PDF layout preservation.
- No cloud sync.
- No multi-user auth.
- No vector DB yet.
- Analysis/chat use local lightweight retrieval first.
- EPUB fixed-layout is not supported.
- UnifiedDocument snapshots do not copy source EPUB/PDF binaries.
- PPTX export is evaluated but not implemented in the initial merge.
