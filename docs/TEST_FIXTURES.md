# Test Fixtures

## Source and License

The EPUB fixtures are generated by `tests/helpers/createEpubFixtures.ts`. They contain synthetic text, minimal XHTML, generated OPF/container files, a tiny placeholder PNG byte sequence, and project-authored CSS. They do not include copyrighted commercial book content.

The tests generate fixtures into a temporary directory at runtime. Large EPUB binaries are not committed.

## Generated Fixtures

- `minimal-epub3.epub`: minimal EPUB 3 with nav and one chapter.
- `epub2-ncx.epub`: EPUB 2 style package with `toc.ncx`.
- `images-and-css.epub`: EPUB with image and stylesheet resources.
- `footnotes-inline.epub`: EPUB with footnote anchors and inline tags.
- `cjk-source.epub`: EPUB with mixed Chinese, Japanese, and English text.
- `malformed-missing-resource.epub`: invalid EPUB with a manifest href that is not present.
- `nested-sections.epub`: multi-file EPUB with chapter, section, subsection headings and a non-linear appendix.
- `split-text-inline.epub`: inline text split across `em`, `strong`, and linked note anchors.
- `entities-special-chars.epub`: XML-sensitive characters, entity references, smart punctuation, and emoji.
- `nav-landmarks.epub`: EPUB 3 nav with toc, landmarks, and page-list; nav is non-linear.
- `duplicate-hrefs.epub`: same filenames in different subdirectories with relative CSS and image resources.
- `large-chapter-chunking.epub`: long synthetic chapter with multiple translatable text-node groups for progress checks.

## Running Compatibility Tests

```bash
npm test
```

The fixture tests use `MockTranslator`, do not call real AI APIs, do not require external EPUBCheck, and do not use the network.

## Supported Scope

Current support targets reflowable EPUB. Fixed-layout EPUB, DRM EPUB, PDF, MOBI, and AZW3 are outside the current scope.

## Alpha Testing Recommendations

- Test with legally obtained personal EPUBs only.
- Keep a backup of original EPUB files.
- Use the built-in validation report first, then optionally configure external EPUBCheck.
- Review exported books in more than one reader when possible.
